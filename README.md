# Excel_Teachers_Upskilling_Dashboard_Project

## Overview
This project presents an interactive Excel dashboard to analyze teacher participation and performance in upskilling programs.

## Objectives
- Track teacher enrollment across courses and states
- Measure completion rates and training effectiveness
- Enable drill-down analysis using filters

## Tools Used
- MS Excel
- Pivot Tables
- Slicers
- Advanced formulas (INDEX-MATCH, IFERROR)

## Key Features
- Interactive KPI cards
- Dynamic insight panel
- Course, state, and mode-based filtering
- Executive-ready dashboard layout

## Insights
- Certain courses show consistently higher enrollment
- Completion rates vary significantly by region
- Online and hybrid modes perform better
